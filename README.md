# 소개
### 임산부 건강관리의 필요성
  * 산모의 평균 연령이 높아짐에 따라 영양 관리 및 운동에 대한 중요성이 더욱 높아짐
  * 임산부는 건강관리가 중요함을 인지하고 있지만 관련 지식에 무지한 경우가 대부분임
### 스마트한 식단 관리와 요가 기능
  * 출산예정일에 기반한 임산부 일일영양섭취기준과 추천 알고리즘을 통한 식단 관리
  * Tensorflow-Lite의 학습된 모델을 이용한 요가자세추정
### 임산부의 편의를 위한 기능
  * 인근 산부인과 검색(위치 기반)과 일정 관리
